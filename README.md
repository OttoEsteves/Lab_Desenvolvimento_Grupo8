# Lab_Desenvolvimento_Grupo8 — Portfólio (HTML/CSS/JS)

Portfólio web estático do **Bernardo Rodrigues**, com seções **Home**, **About me**, **Programming Experience**, **Skills**, **Projects** e **Contact**.

---

## ✨ O que tem no projeto

- Menu lateral com navegação por âncoras (Home / About me / Skills / Projects / Contact)
- Efeitos/animações de exibição (título inicial e seção About me)
- Seção de currículo com link para `curriculo.pdf`
- Seção de projetos com links para repositório e demo (quando disponível)
- Formulário de contato integrado ao **StaticForms** + confirmação ao enviar e feedback no botão

---

## 🧰 Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap 5 (CDN)
- Bootstrap Icons (CDN)

---

## 📁 Estrutura (arquivos principais)

- `index.html`
- `style.css`
- `js.js`

- Imagens utilizadas no layout (referenciadas no `index.html`)
